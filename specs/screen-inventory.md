# Screen Inventory

## 1. Onboarding / Welcome

- Value proposition with short, clear messaging
- Simple intro slides or a single welcome screen
- Start immediately CTA
- Optional sign in / continue as guest
- Establish trust with calm fintech visuals

## 2. Account Setup

- Choose currency
- Set starting balance
- Create first account / payment source
- Optional monthly income estimate
- Optional budgeting preference
- Success state when setup is completed

## 3. Home / Dashboard

- Total balance
- Income this month
- Expenses this month
- Savings this month
- Budget status
- Spending by category preview
- Recent transactions
- Smart insight cards
- Quick actions
- Weekly / monthly toggle
- Cash flow mini chart
- Strong floating action button for new transactions

## 4. Full Transactions Screen

- Full transaction list
- Search bar
- Filters for date, account, category, type, and tags
- Sort options
- Group by day, week, or month
- Clear visual distinction between income and expense
- Empty state when there are no transactions

## 5. Add Transaction Screen

- Fast entry flow
- Toggle for expense / income / transfer
- Amount as the primary field
- Category selector
- Account selector
- Date and time
- Note field
- Tags
- Receipt attachment placeholder
- Recurring toggle
- Split transaction option
- Save button reachable with one hand

## 6. Transaction Details Screen

- Full breakdown of the transaction
- Category, account, amount, date, note, tags
- Recurrence info
- Attachment / receipt area
- Actions: edit, duplicate, delete

## 7. Categories Screen

- Category list with icon, color, budget, and spending summary
- Default expense categories:
  - Food
  - Transport
  - Rent
  - Shopping
  - Entertainment
  - Health
  - Bills
  - Education
  - Travel
  - Savings
  - Salary
  - Freelance
  - Investments
  - Gifts
  - Other
- Create custom category
- Edit category
- Archive or hide category
- Differentiate income categories from expense categories

## 8. Create / Edit Category Screen

- Category name
- Icon picker
- Color picker
- Category type: income or expense
- Optional budget assignment
- Save and cancel states

## 9. Budgets Overview Screen

- Total monthly budget
- Remaining budget
- Progress by category
- Warning states near the limit
- Over-budget states
- Motivating visual summary
- Clear CTA to edit budgets

## 10. Category Budget Detail Screen

- Budget amount
- Spent so far
- Remaining amount
- Forecast / expected end-of-month state
- Related transactions
- Option to adjust budget

## 11. Reports / Analytics Screen

- Spending by category
- Income vs expense
- Cash flow over time
- Trend charts
- Weekly / monthly / yearly tabs
- Custom date range
- Most spent category
- Average daily spend
- Net savings trend
- Clean visual hierarchy without overload

## 12. Accounts Screen

- Multiple account cards:
  - cash
  - checking
  - savings
  - debit card
  - credit card
  - digital wallet
- Balance per account
- Total across all accounts
- Quick transfer action
- Quick add account action

## 13. Account Detail Screen

- Balance history
- Transactions for that account
- Account type
- Edit account
- Archive account

## 14. Transfer Between Accounts Screen

- From account
- To account
- Amount
- Date
- Note
- Confirmation state

## 15. Recurring Transactions Screen

- List of recurring income and expense items
- Upcoming recurring items
- Frequency options: daily, weekly, monthly, yearly
- Active / inactive toggle
- Edit recurring rule

## 16. Bills & Subscriptions Screen

- Upcoming bills
- Subscription list
- Due dates
- Amounts
- Paid / unpaid state
- Reminder option

## 17. Savings Goals Screen

- Create savings goals
- Goal name
- Target amount
- Current progress
- Deadline
- Contribute to goal action
- Visual progress cards

## 18. Notifications / Alerts Screen

- Budget limit warnings
- Upcoming bills
- Recurring reminders
- Unusual spending insight
- Goal milestones

## 19. Export / Data Screen

- Export to CSV or PDF
- Select date range
- Select accounts and categories
- Confirmation state

## 20. Settings Screen

- Currency
- Theme
- Dark mode
- Notification preferences
- Data backup / sync placeholder
- Security settings
- Language
- Help / support

## 21. Profile / Preferences Screen

- Basic user info
- Finance preferences
- Start-of-month preference
- Budgeting style

## Critical states

These states must be represented across the main screens:

- Empty state
- Loading state
- Error state
- Success state
- Confirmation modal
- Delete confirmation
- Budget exceeded warning
- No results search state
